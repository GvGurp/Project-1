Office 365

Office 365 is een verzameling van meerdere internetdiensten.De verzamling is bedoeld voor thuisgebruik, bedrijven en het onderwijs.Je kunt de nieuwste productiviteits-apps zoals word, excel,powerpoint, onedrive, Microsoft teams en nog veel meer krijgen als je office 365 hebt.

Techniek college rotterdam biedt office 365 voor ieder student zolang ze daar studeren. Je kan je account activeren met jouw schoolmail en wachtwoord.Niet gelukt?Ga dan naar jouw mentor of neem contact op met de techniek college info punt.

Visual studio code

Visual studio code is een broncode-editor ontwikkeld door microsoft voor Linux, macOS en windows.Visual studio code bevat ondersteuning voor ingebouwde GIT-controle, syntax highlighting, codefragmenten,debugging, intelligente code-aanvulling en code refactoring.

In visual code kan je code wegschrijven in meerdere programmeertalen.Je kan bijvoorbeeld websites maken met HyperText Markup Language (Html) en Cascading Style Sheets (Css). Ook kun je een game maken in visual studio code met JavaScript (JS).

Visual studio code programmeertalen:
Javascript (JS),HyperText Markup Language (Html), C#,Cascading Style Sheets (Css), Typescript.
