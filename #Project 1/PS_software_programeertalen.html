HyperText Markup Language

HTML is een programmeertaal en staat voor HyperText Markup Language.Met gebruik van HTML kan je webpaginaâ€™s schrijven (coderen).

De bouw van een HTML-pagina is relatief eenvoudig, want de volgorde van de code is altijd hetzelfde.De HTML-pagina begint met een DOCTYPE-declaratie.De DOCTYPE-declaratie geeft aan welke HTML-versie er wordt gebruikt.Daarna komt de HEAD.In de HEAD-element wordt algemene informatie over de document geplaatst.Ten slot komt de BODY-element. Waar de inhoud van het document staat wordt aangeven met de BODY-element. Ook kan de basisweergave zoals achtergrond kleuren, tekst kleuren en groottes in de BODY-element staan.

Cascading Style Sheets

CSS is een programmeertaal en staat voor Cascading Style Sheets. Met gebruik van CSS kan de opmaak van html bestanden bewerken.Je moet dan in je HTML-pagina naar de stylesheet (CSS) verwijzen.

Je kan elk HTML bestand apart aanspreken in de CSS.Hier wordt vaak identificeerders (id=) en classes (class=) voor gebruikt.Met identificeerders en classes kun je dezelfde HTML-elementen op verschillenden manieren vormgeven.Het maakt dan ook niet uit waar de HTML-element op de website staat.

Javascript

JS is een scripttaal en staat voor Javascript.Met gebruik van javascript kan je een website interactief maken.

Javascript staat in een HTML-code.Dat kan je op twee manieren doen.De eerste manier is via een script tag.De script tag zit in de HTML-pagina zelf.Een nadeel van deze manier is dat je HTML-pagina heel lang kan worden.De tweede manier is via een externe code.In de HTML-pagina moet je dan naar jouw externe code verwijzen.
